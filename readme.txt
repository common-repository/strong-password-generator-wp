WordPress Strong Password Generator is a plugin for WordPress to generate strong and random passwords.

== Description ==

WordPress Strong Password Generator is a plugin for WordPress to generate strong passwords of length 6 to 256. You can choose to include capital letters, small letters, number and special characters in the password. This plugin also allows the generated password to be copied to the clipboard.

= Major Features =

* Add widget with shortcode anywhere in the site.
* Easy to use widgets available.
* Visitor can choose to include small letters, capital letters, numbers and special characters in password.
* Password Length can be chosen.

== Installation ==

Download WP Strong Password Generator from WordPress repository and install it in your website and activate the plugin. After activating, you can display the WP Strong Password Generator by adding the shortcode [strong-password-generator] or visit the Widgets section and Select Strong Password Generator to add it to your WordPress website. 

Shortcode: 
[strong-password-generator]
